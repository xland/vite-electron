# vitetron

Develop and build electron project with vite and vue3

Install

> $ yarn create vite-app <project-name>
> $ cd <project-name>

```bash
yarn add vitetron --dev
```

add scripts to your package.json

```json
  "scripts": {
    "start": "vitetron start",
    "release": "vitetron release"
  }
```

run `yarn start`
